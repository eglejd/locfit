LocFit
Scored Website Localization QA
Add the key pages of a site, get one ranked view of where its localization is breaking.

Most localization isn't meant to be perfect. It's meant to be good enough for the job. LocFit grades each page with the MQM framework, ranks them worst-first, and separates what's broken from what's just polish.

Pages to review
Set the target language for each page in plain words: "Lithuanian", or a locale code like "lt-LT", both work. Each page is reviewed separately with the same MQM engine; the overview ranks the site’s pages worst-first. To save a live page as a file: open it, Ctrl/Cmd+S → "Webpage, HTML only".
